# تفاصيل واجهات المستخدم والتقنيات

## 1. تسجيل الدخول والإعداد الأولي

### شاشة الترحيب
- شعار التطبيق مع رسالة ترحيبية
- زر "تسجيل جديد" و "تسجيل دخول"
التقنيات:
- Firebase Authentication
- JWT للجلسات

### نموذج التسجيل (3 خطوات)
#### الخطوة 1: معلومات الحساب
- حقل البريد الإلكتروني
- حقل كلمة المرور
- حقل رقم الجوال (مع رمز التحقق)
التقنيات:
- Twilio للرسائل النصية
- التحقق من صحة البريد

#### الخطوة 2: معلومات الشركة
- اسم الشركة
- العنوان
- عدد المركبات
- نوع النشاط (قائمة منسدلة)
التقنيات:
- Google Places API للعناوين
- Cloud Firestore للتخزين

#### الخطوة 3: إضافة المركبات
- نموذج متكرر لكل مركبة:
  * رقم اللوحة
  * النوع/الموديل
  * سعة الوقود
  * رفع المستندات
التقنيات:
- Cloud Storage لتخزين المستندات
- OCR لقراءة بيانات المستندات

## 2. الشاشة الرئيسية (Dashboard)

### القسم العلوي
- شريط التنقل الرئيسي
- ملخص سريع (عدد المركبات النشطة/المتوقفة)
- تنبيهات عاجلة

### خريطة تفاعلية (القسم الأكبر)
- موقع جميع المركبات
- حالة كل مركبة (متحركة/متوقفة)
- معلومات سريعة عند النقر
التقنيات:
- Google Maps API
- WebSocket للتحديثات المباشرة
- Geolocation API

### قائمة المركبات (الجانب)
- قائمة قابلة للتمرير
- لكل مركبة:
  * رقم اللوحة
  * اسم السائق
  * الحالة
  * مستوى الوقود
  * السرعة الحالية

## 3. صفحة تفاصيل المركبة

### معلومات أساسية
- صورة المركبة
- البيانات الأساسية
- حالة التشغيل

### قسم الوقود
- مؤشر مستوى الوقود الحالي
- رسم بياني للاستهلاك
- سجل التعبئة
التقنيات:
- Chart.js للرسوم البيانية
- IoT Integration لحساسات الوقود

### سجل الحركة
- جدول زمني للتحركات
- خريطة المسار
- نقاط التوقف

## 4. نظام التنبيهات

### أنواع التنبيهات
- تنبيهات فورية (Push Notifications)
- تنبيهات داخل التطبيق
- تنبيهات بريدية
التقنيات:
- Firebase Cloud Messaging
- SMTP للبريد
- WebSocket للتحديثات المباشرة

## التقنيات الأساسية المطلوبة:
1. Backend:
   - Node.js/Express
   - MongoDB/Firestore
   - Redis للتخزين المؤقت
   - WebSocket Server

2. Frontend:
   - React Native (تطبيق موحد)
   - Redux لإدارة الحالة
   - MapBox/Google Maps
   - Socket.io-client

3. خدمات:
   - Firebase
   - AWS/Google Cloud
   - Twilio
   - Payment Gateway

4. أمان:
   - JWT
   - SSL/TLS
   - تشفير البيانات
   - Rate Limiting
